one-value syntax the single value sets the width of all four border sides.

two-value syntax the first value sets the width of top and bottom while the second value sets the width of left and right.

three-value syntax the first value sets the top border,the second value sets the left and right border, the third value sets the bottom border.

four-value syntax each value sets the width of border individually in the order of top, right, bottom, left.

Note:You must declare the border-style property before the border-width property.
